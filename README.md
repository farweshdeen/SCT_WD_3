# Tic Tac Toe Game

SkillCraft Technology - Web Development Internship

## Task 03: Tic Tac Toe Game (Alternative Task)

### Project Description

This project was developed as part of **Task 03** of the **SkillCraft Technology Web Development Internship**.

The objective of this task was to design and develop an interactive Tic Tac Toe game using **HTML, CSS, and JavaScript**. The game allows two players to take turns marking spaces on a 3×3 grid, automatically detects the winner or a draw, and provides an option to restart the game for a new match.

### Features

* Interactive 3×3 Tic Tac Toe game
* Two-player gameplay
* Automatic winner detection
* Draw detection
* Restart game functionality
* Responsive design for desktop and mobile devices
* Clean and modern user interface

### Technologies Used

1. HTML5
2. CSS3
3. JavaScript
